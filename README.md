# interview-feedback-app

# 回答フィードバックWebアプリ

## 概要
問題をDB内から選択し、それに対する回答を音声回答（Whisperによって文字起こし）or キーボード入力を行い、
Geminiを用いて回答内容の改善点やフィードバックを提示するWebアプリケーションです。

## 開発背景
学部内カリキュラムの臨地実務実習でお世話になった企業様側からいただいた課題です。
音声認識機能と生成AIを掛け合わせたシステムは今後需要が増えていくと感じ作成しました。

## 使用技術
- Python
- Flask（またはFastAPI）
- HTML/CSS
- JavaScript
- OpenAI Whisper
- Google Gemini API

## 主な機能
- 音声の文字起こし
- AIによる回答内容の評価
- 改善点の提示

## 工夫した点
Whisperの認識精度向上のために複数の手法を比較し、
最も精度の高い方法を採用した。

また、Geminiへのプロンプトを調整することで、
具体的な改善点が得られるように工夫した。
